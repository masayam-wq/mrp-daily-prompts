# Weekly Review実行指示

以下のWeekly入力を用いて、医学論文の週次レビューを日本語で作成してください。

## 必須条件

1. Dropboxコネクタは使わない。
2. このURL本文に含まれるWeekly入力だけを対象にする。
3. JSONは出力しない。machine_readable_summary も出力しない。
4. PubMed/DOI/Source URLがある場合は、必要に応じてWeb/学術検索で一次情報を確認してよい。
5. 確認できた情報と、入力情報のみからの解釈を分ける。
6. 個別患者への診療助言・治療推奨として書かない。

## 出力構成

1. 今週の結論
2. 今週の最重要論文 Top10
3. ジャンル別レビュー
4. 今週のトレンド
5. 安全性・限界
6. 研究・診療・発信に使えそうなテーマ
7. 来週ウォッチすべきキーワード
